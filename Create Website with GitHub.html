ChatGPT said:

Creating a website with GitHub is straightforward, especially with GitHub Pages, which allows you to host static websites directly from a repository. Here's a step-by-step guide:

✅ Step 1: Create a GitHub Account

If you don’t have one yet, go to https://github.com and sign up.

✅ Step 2: Create a New Repository

  1. Go to your GitHub profile.

  2. Click the + icon in the top right corner → select "New repository".

  3. Name your repository something like my-website.

  4. Choose Public and check "Add a README file" (optional but helpful).

  5. Click Create repository.

✅ Step 3: Enable GitHub Pages

  1. Go to your repository’s main page.

  2. Click the "Settings" tab.

  3. Scroll down to Pages (in the left sidebar or bottom depending on UI).

  4. Under "Source", choose main (or master) branch and /root folder.

  5. Click Save.

GitHub will generate a URL like:

perl
https://your-username.github.io/my-website/

✅ Step 4: Add Your Website Code

You can add HTML/CSS/JS files in two ways:

Option A: Upload Files Directly

  1. On your repo page, click "Add file" → "Upload files".

  2. Upload an index.html file (this is your main page).

  3. Click "Commit changes".

Option B: Use Git Locally (Advanced)

  1. Install Git.

  2. Clone your repo:

    bash
    git clone https://github.com/your-username/my-website.git

  3. Move into the directory:

    bash
    cd my-website

  4. Create or edit index.html and any other files.

  5. Add, commit, and push:

    bash
    git add .
git commit -m "Add website files"
git push

✅ Step 5: Visit Your Website

After a minute or so, visit:

perl
https://your-username.github.io/my-website/

You should see your website live!

✅ Sample index.html

Here’s a simple HTML file you can use to test:

html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0"/>
  <title>My GitHub Website</title>
</head>
<body>
  <h1>Hello from GitHub Pages!</h1>
  <p>This is my first website hosted on GitHub.</p>
</body>
</html>

    html
